## 介绍
- Mirs是一个采用c++编写，基于事件驱动的小型Web服务器，支持GET、POST请求以及动态网页的处理。
## 运行环境
- 系统：ubuntu19.10
- 处理器：Intel® Core™ i5-7200U CPU @ 2.50GHz × 4
- 内存：7.7 GiB
- 编译器：gcc version 9.2.1

## 安装及运行
运行前请先安装CMake（2.8及以上版本）
```
cd server
cmake .
make
./Server
```
